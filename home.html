Welcome
To the UCLA PEEC Lab Toolbox

Guidelines and tool storage local website

First of all, welcome to this site, ideally this will be hosted on a local server. This site is designed to be a toolbox for the PEEC lab at UCLA, it contains some useful links, tools, guides. It is not meant to be a public website, but rather a local server for the lab members to access some useful information. As previouslly discussed with the professor he do not want too many internal information to be disclosed to the public, but some information still needs to stay in the form of a webpage for the ease of access, (slack does not allow access to information more than 90 days old unless you pay them and you also need to download them just to read them), hence this local server is established for us to access some useful links, tools, guides, and information.
